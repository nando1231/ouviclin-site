# Ouviclin - Site Estático

Este repositório contém os arquivos estáticos do site Ouviclin, extraídos para migração ou backup.

## Estrutura de Arquivos
- `index.html`: Página principal do site (contém CSS e JS inline).
- `images/`: Pasta contendo todos os recursos visuais (imagens, ícones, logotipos).

## Como usar no WordPress
Para utilizar este site em uma hospedagem WordPress, você tem as seguintes opções:

1. **Página Estática em Subdiretório**: Faça o upload destes arquivos para uma pasta no seu servidor (ex: `/public_html/site-estatico/`).
2. **Template Customizado**: Integre o código do `index.html` em um tema do WordPress.
3. **Plugin de Importação HTML**: Utilize plugins como "HTML Import 2" para importar o conteúdo.
